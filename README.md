# White_Board

## 服务器端开启方式：

**在根目录下运行以下两个命令：**

python3 manage.py runserver 0.0.0.0:8000

daphne -b 0.0.0.0 -p 5015 White_Board.asgi:application

**网页客户端打开方式**

url：http://123.57.187.239:8000/
